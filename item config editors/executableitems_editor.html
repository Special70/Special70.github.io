For Loop
Start Value: This is where you will be inputting the
first value of your for loop.

Increment Value: This is where you will be inputting the
increasing amount. If the Start Value or
the Increment Value has one non-numerical
character, the letters stack instead.

Increment Amount: This is where you will be inputting the
amount of elements you would want in your
FOR loop. Typing 0 will give you a blank
FOR loop. Typing 3 will give you a FOR
loop with 3 elements in it.
